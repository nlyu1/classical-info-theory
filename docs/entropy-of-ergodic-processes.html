12 Entropy of Ergodic Processes

In this section, we consider random variables over a common space \((\Omega, \Sigma)\).

Preliminaries

Definition 12.1 (measure-preserving transformation) \(\tau:\Omega\to \Omega\) is a measure preserving transformation, or probability preserving transformation (p.p.t.) if \[ \mathrm{Pr}(\forall E\in \Sigma) = \mathrm{Pr}(\tau^{-1}E) \] A set \(E\) is \(\tau\)-invariant if \(E=\tau^{-1}(E)\). The collection of all \(\tau\)-invariant sets form a \(\sigma\)-algebra \(\Sigma_{\mathrm{inv}}\).

Definition 12.2 (stationary process) A random process consisting of a sequence of random varables \(\{S_1, S_2, \cdots\}\) (of signature \(\Omega\to \mathbb R\)) is stationary if there exists a measure-preserving \(\tau:\Omega\to \Omega\) such that \(S_j = S_0\circ \tau^j\).

One may take this definition to the joint distribution and alternatively declare \(\{S_1, \cdots\}\) stationary if the joint distribution is invariant to shifts in time; in this case the sample space corresponds to the (infinite) joint sequence \(\Omega=\mathcal S^{\infty}\) with p.p.t. defined as \[ (s_0, s_1, \cdots) \xrightarrow \tau (s_1, s_2, \cdots) \]

An event \(E\in \Sigma\) is shift-invariant if \(E=\tau^{-1} E\) for the shift-operator \(\tau\).

Definition 12.3 (tail algebra) The tail \(\sigma\)-algebra contains events invariant to any finite number of initial changes \[ \Sigma_{\mathrm{tail}} = \bigcap_{n\geq 1} \sigma(S_n, S_{n+1}, \cdots) \] Here \(\sigma(S_n, S_{n+1}, \cdots)\) denotes the collection of events which can be described in terms of these random variables. More precisely, let \(V\) be the codomain of \(S_j\) and \(\mathcal B(V)\) the Borel algebra, \[ \sigma(S_n, S_{n+1}, \dots) = \sigma\left(\left\{ S_n^{-1}(B) \mid B \in \mathcal{B}(V) \right\}, \left\{ S_{n+1}^{-1}(B) \mid B \in \mathcal{B}(V) \right\}, \ldots \right). \]

Proposition 12.1 (shift-invariant events and tail algebra) All shift-invariant events belong to a strict subset of tail events \(\Sigma_{\mathrm{inv}}\subsetneq \Sigma_{\mathrm{tail}}\). Consider, for example, \(\{\exists n:x_0=0, \,\,\forall \text{ odd }i\geq n\}\).

Proposition 12.2 (Poincare recurrence) Given a measure-preserving \(\tau\), for any measurable \(A\) with \(\mathrm{Pr}[A]>0\) we have \[ \mathrm{Pr}\left[ \bigcup_{k\geq 1} \tau^{-k} A \,\bigg|\, A \right] = \mathrm{Pr}[\tau^k(\omega\in A)\in A\text{ occurs infinite often } | A] = 1 \]

Proof Let \(B = \bigcup \tau^{-k}A\), by Bayes it suffices to show that \(\mathrm{Pr}[A\cap B]=\mathrm{Pr}[A]\mathrm{Pr}[B|A]=\mathrm{Pr}[A]\), which is equivalent to \(\mathrm{Pr}[A\cup B]=\mathrm{Pr}[B]+\mathrm{Pr}[A\cap \neg B]=\mathrm{Pr}[B]+\mathrm{Pr}[\neg B|A]_{=0}\mathrm{Pr}[A] = \mathrm{Pr}[B]\). Recall that \(\tau^{-1}\) is the forward-evolution map. Notice \[ B = \tau^{-1}A \cup \tau^{-1}B = \tau^{-1}(A\cup B)\implies \mathrm{Pr}[B] = \mathrm{Pr}[\tau^{-1}(A\cup B)] = \mathrm{Pr}[A\cup B] \]

Example 12.1 (physical recurrence) The phase space flow in classical Hamiltonian mechanics is a measure-preserving transform by Liouville’s theorem. Consider a system of two cylinders of gas immediately before they’re allowed to mix with \(A\) the set of all separated states, and \(\omega\in A\) an initial separated state. Poincare recurrence states that all molecules in each cylinder will exactly return to their cylinder (although not necessarily to their initial state) infinitely often. Thermodynamically, however, this will take unrealistically long.

Definition 12.4 (ergodicity) \(\tau\) is ergodic if \(\forall E\in \Sigma_{\mathrm{inv}}\), we have \(\mathrm{Pr}[E]=0\) or \(\mathrm{Pr}[E]=1\). In other words, all shift-invariant events are deterministic.
